# 特約まる見えチェッカー

加入中の保険の特約を3分でチェックし、重複している特約と削減目安額を表示するWebツール。

## 使い方

1. ブラウザで [https://yuki-sato.github.io/rider-checker/](https://yuki-sato.github.io/rider-checker/) にアクセス
2. 加入中の保険を選択
3. 各特約に「付いている／不明／付いていない」を回答
4. 重複している特約と月額削減目安が表示されます

## 特徴

- 完全ブラウザ完結（入力データは一切サーバーに送信されません）
- 3分で完了
- 個人賠償特約・弁護士費用特約・旅行傷害特約・先進医療特約など6グループの重複を自動検出
